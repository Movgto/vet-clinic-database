### Key Features <a name="key-features"></a>

- **Database created with PostgreSQL.**
- **Created a table to store animal data structured with: name, date of birth, neutered status, and escape attempts.**
- **SQL queries used to select objects from the table based on conditions and parameters, and also select only some fields of the data structure based on the corresponding name.**

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need to have `PostgreSQL` installed in your system.

You can download the program [here](https://www.postgresql.org/download/)
### Setup

1. Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone https://github.com/Movgto/vet-clinic-database
```

2. Paste the code from the `.sql` files on the `PostgreSQL` terminal to create the `animals` table, apply the corresponding modifications, and display specific information from the table using queries.
